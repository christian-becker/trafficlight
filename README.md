# trafficlight
build a traffic light with Raspberry Pi (Zero) and Pimoroni Blinkt! :vertical_traffic_light:

Just a simple example of what you can do with Pimoroni Blinkt! on top of the GPIO pins of Raspberry Pi (Zero).
